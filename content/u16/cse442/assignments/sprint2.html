== Version Control and Project Management (1 Point)

Proper use of version control with professionalism. A meaningful branching model is being used. Commit messages generally make sense.

Organization and progress made on tasks. User stories are recorded and each task clearly relates to a story and is assigned to a group member(s). Must show reasonable progress on tasks to earn this point.

This point will not be awarded if there hasn't been meaningful progress made on the project.

== Individual Participation (1 Point)

You are attending group meetings and making meaningful progress on tasks assigned to you. This will be determined by checking individual commits and tasks to which each group member is assigned as well as taking attendance at all meeting with course staff. These and any other relevant information will be used to determine if this point will be awarded.

This point is graded on an individual basis.

== Software Delivered (2 point)

Deliver working software with some functionality. In addition to showing the the user the look and feel of the software there will be some meaningful functionality.
